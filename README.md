# Grandpaan_resume

利用React框架实现
项目预览
-------
- [预览地址](https://grandpaan.github.io/Grandpaan_resume/)


如何开始？
-------
```
git clone git@github.com:GrandpaAn/Grandpaan_resume.git
cd Grandpaan_resume
npm install
npm run dev
```

然后访问：
```
http://127.0.0.1:8080/
```

如何生成生产环境项目进行部署？？
-------
```
git clone git@github.com:GrandpaAn/Grandpaan_resume.git
cd Grandpaan_resume
npm install
npm run build

```

然后你复制目录build下的
```
- bundle.js
- index.html
```
到你的服务器中去
